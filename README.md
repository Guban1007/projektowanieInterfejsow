# TaskFlow - Task Management App

Projekt frontendowy na przedmiot **Projektowanie Interfejsów Użytkownika**.

## Zakres funkcjonalny

- dodawanie, edycja i usuwanie zadań,
- oznaczanie zadań jako wykonane,
- lista zadań z responsywnym interfejsem,
- filtrowanie: wszystkie, aktywne, wykonane, na dziś,
- sortowanie: najnowsze, termin, priorytet, tytuł,
- priorytety, terminy i tagi,
- wyszukiwarka,
- tryb ciemny,
- walidacja formularza,
- zapis danych w `localStorage`.

## Uruchomienie

1. Rozpakuj archiwum.
2. Otwórz plik `index.html` w przeglądarce.
3. Aplikacja działa bez backendu i bez instalowania zależności.

## Struktura

- `index.html` - struktura widoków aplikacji,
- `styles.css` - wygląd, responsywność, tryb ciemny,
- `app.js` - logika CRUD, filtry, sortowanie i localStorage,
- `dokumentacja-projektowa.pdf` - dokumentacja UX/UI.
